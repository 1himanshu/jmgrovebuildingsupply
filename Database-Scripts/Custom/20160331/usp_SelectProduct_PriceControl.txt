

-- =============================================































































-- Author:		<Author,,Name>































































-- Create date: <Create Date,,>































































-- Description:	<Description,,>































































-- =============================================































































Create PROCEDURE USP_SelectProduct_PriceControl 































































(































































	































   @ProductId int=0,















   @Type VARCHAR(5) = '',















   @ProductName VARCHAR(50) = ''















  































)































































AS































































BEGIN































































	-- SET NOCOUNT ON added to prevent extra result sets from































































	-- interfering with SELECT statements.































































	SET NOCOUNT ON;















































































	if(@Type='S')















	begin















	Select * FROM [dbo].[tblProductMaster] where ProductId=@ProductId















	end































	if(@Type='D')















	begin















	delete from [dbo].[tblProductMaster] where ProductId=@ProductId















	end















































	if(@Type='Dup')















	begin















	Select * FROM [dbo].[tblProductMaster] where ProductName=@ProductName















	end















































	if(@Type='DupU')















	begin















	Select * FROM [dbo].[tblProductMaster] where ProductName=@ProductName and ProductId != @ProductId















	end















































END
